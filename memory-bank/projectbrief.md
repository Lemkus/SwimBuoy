# Project Brief — SwimBuoy

Обновлено: 2026-06-08

## Цель

Тренировки на открытой воде с **виртуальными буями** (без физических меток) на Garmin FR955.

## MVP

- Connect IQ watch-app: крупные **метры** до активной точки + **стрелка** (bearing).
- Навигация к точке (`point_proximity`), не отклонение от линии курса.
- Засчитывание буя: `arrivalRadiusM` + `dwellSec`.
- В заплыве показывается только **текущая** цель.

## Не в MVP

Bluetooth в воде, геймификация, ghost, миссии, публикация в Connect IQ Store.

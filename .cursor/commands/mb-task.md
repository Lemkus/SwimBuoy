Создай или обнови активную задачу в `memory-bank/tasks.md`.

**Ввод**: название или описание задачи после `/mb-task`.

**Шаги**

1. Если ввод пустой — спроси, какую задачу отслеживать.

2. Открой `memory-bank/tasks.md`:
   - если секция задачи есть — обнови
   - иначе добавь новую секцию в конец

3. В секции задачи должны быть:
   - **Goal** — зачем это SwimBuoy
   - **Scope** — что входит / не входит
   - **Checklist** — шаги (`- [ ]`)
   - **Verification** — как проверить (симулятор, FIT, поле)
   - **Status**: `planned` | `in_progress` | `blocked` | `done`

4. Синхронизируй контекст:
   - обнови `memory-bank/activeContext.md` — эта задача активна
   - при необходимости добавь пункт в `docs/status.md` → «В процессе / следующий шаг»

5. Верни:
   - какая секция создана/обновлена
   - сколько открытых пунктов чеклиста
   - рекомендуемое следующее действие

**Ограничения**
- `memory-bank/tasks.md` — source of truth для активной работы.
- Одна явно помеченная активная задача, если пользователь не просит иначе.
- Архивацию делай через `/mb-archive`, не здесь.
